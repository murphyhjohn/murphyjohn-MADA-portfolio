dat1 <- dat %>%
  select(
    years = Survey.Years,
    sex = Sex,
    age = Age.Group,
    measure = Measure,
    percent = Percent
  ) 
# filter for more granular data - separate by sex, age categories
dat2 <- dat1 %>%
  filter(
    sex != "All",
    age != "20 and over",
    age != "2-19"
  ) %>%
  mutate(
    age_factor = factor(age, levels = c(
      "2-5","6-11","12-19","20-29","30-39","40-49",
      "50-59","60-69","70 and over")
    ),
    age = as.numeric(age_factor),
    sex = factor(sex, levels = c("Female", "Male")),
    years_factor = factor(years, levels = c(
      "1999-2000","2001-2002","2003-2004",
      "2005-2006","2007-2008","2009-2010",
      "2011-2012","2013-2014","2015-2016","2017-2018"
    )),
    years = as.numeric(years_factor)
  )

# turn obesity, cholesterol, and hypertension into their own variables
dat3 <- dat2 %>%
  group_by(years, sex, age) %>%
  tidyr::pivot_wider(
    names_from = measure,
    values_from = percent
  )

EDA Conclusions

There are 10 2-year years categories from 1999-2018. For each years category, there is an observation for each sex (Female, Male), and within each sex, an observation for each of 6 age groups (20-29,30-39,40-49,50-59,60-69,70 and over). This makes \(6*2=12\) observations each year and \(10*12=120\) total observations in the data. Each observation has a value for percentage of population with obesity, high cholesterol, and hypertension.

Obesity overall has mean 35.31 with (30.57, 40.90) quartiles. For every one unit increase in year group, there is a \(1.2\) unit increase in obesity. Male sex has \(3.03\) unit decrease in obesity from female sex. Excluding the final age category, for every subsequent age group, there is a \(3.14\) unit increase in obesity. There is no difference in obesity between age categories “20-29” and “70 and over”.

Hypertension overall has mean 49.98 with (30.65, 70.90) quartiles. There is no obvious relationship between hypertension and year or sex. For every subsequent age group, there is a \(13.56\) unit increase in hypertension.

Cholesterol overall has mean 14.35 with (8.5, 18.55) quartiles. For every one unit increase in year, there is a \(0.9\) unit decrease in cholesterol. Male sex has \(3.45\) unit decrease in cholesterol from female sex. For every subsequent age group, there is a \(1.64\) unit increase in cholesterol.
